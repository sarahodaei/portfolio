# Sara Hodaei — Portfolio

Personal portfolio website for Sara Hodaei, Data Scientist. Built with React + Vite and deployed on Vercel at [sarahodaei.com](https://www.sarahodaei.com).

---

## Tech Stack

- **React** — component-based UI
- **Vite** — build tool and dev server
- **Tailwind CSS** — utility-first styling
- **Vercel** — deployment and hosting

---

---

## Features

- **Intro animation** — Logo fades in, shrinks into a Star Wars-style perspective crawl with a personal bio, then loops. Timings are responsive: 40s on desktop, 25s on mobile.
- **Navbar** — Fixed, transparent on load, darkens on scroll. Includes a music toggle and an availability badge. Links scroll to all sections.
- **Projects** — 6 project cards in a responsive grid (3 columns → 2 → 1). Each card shows project type, title, description, tech tags, date, and a GitHub link where available.
- **Skills** — Grouped into Languages, ML/DL, NLP, Data & Analysis, and Databases. Each group has a colour-coded icon and pill-style tags.
- **Star field** — Animated star background across the full page.
- **Music** — Optional background theme toggled from the navbar.
- **Responsive** — All sections adapt from mobile (320px) to large desktop.

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
